# Модуль списка авторов (Authors)

## Описание

Модуль отображает список всех авторов в алфавитном порядке с количеством книг каждого автора.

## Архитектура

### Серверная часть (PHP)

**Файл:** `authors.class.php`

#### Основные функции

1. **Получение авторов:**
   - Загружает список авторов из view `v_author`
   - Группирует по первой букве имени
   - Сортирует по алфавиту

2. **Группировка:**
   - Группирует авторов по первой букве
   - Создает структуру для отображения

#### Структура данных

```php
$groupedAuthors = [
    'А' => [
        ['name' => 'Автор 1', 'count' => 10],
        ['name' => 'Автор 2', 'count' => 5],
    ],
    'Б' => [
        ['name' => 'Автор 3', 'count' => 8],
    ],
    // ...
]
```

### Клиентская часть

**Файл:** `authors.tpl`

#### Отображение

1. **Заголовок:** "Авторы"
2. **Группы авторов:**
   - Заголовок с буквой
   - Сетка авторов
   - Каждый автор - ссылка на страницу автора
   - Отображение количества книг

## Использование

### URL структура

```
/authors/
```

### Переход на страницу автора

Клик по автору ведет на `/author/?author_name={name}`

## Файлы

- `authors.class.php` - Серверная логика
- `authors.tpl` - Шаблон страницы

## Зависимости

- View БД: `v_author`
- Модуль `author` - для отображения страницы автора

## Особенности

1. **Алфавитная группировка:** Авторы сгруппированы по первой букве
2. **Счетчики:** Отображение количества книг каждого автора
3. **Навигация:** Ссылки на страницы авторов
