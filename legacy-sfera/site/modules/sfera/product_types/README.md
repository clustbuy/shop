# Модуль списка типов товаров (Product Types)

## Описание

Модуль отображает список всех типов товаров в алфавитном порядке с количеством товаров каждого типа.

## Архитектура

### Серверная часть (PHP)

**Файл:** `product_types.class.php`

#### Основные функции

1. **Получение типов товаров:**
   - Загружает список типов из view `v_tip_tovara`
   - Группирует по первой букве названия
   - Сортирует по алфавиту

2. **Группировка:**
   - Группирует типы по первой букве
   - Создает структуру для отображения

#### Структура данных

```php
$groupedProductTypes = [
    'А' => [
        ['id' => 1, 'value' => 'Тип 1', 'cnt' => 10],
        ['id' => 2, 'value' => 'Тип 2', 'cnt' => 5],
    ],
    'Б' => [
        ['id' => 3, 'value' => 'Тип 3', 'cnt' => 8],
    ],
    // ...
]
```

### Клиентская часть

**Файл:** `product_types.tpl`

#### Отображение

1. **Заголовок:** "Типы товаров"
2. **Группы типов:**
   - Заголовок с буквой
   - Сетка типов
   - Каждый тип - ссылка на страницу типа
   - Отображение количества товаров

## Использование

### URL структура

```
/product_types/
```

### Переход на страницу типа

Клик по типу ведет на `/product_type/?product_type_id={id}&page=1`

## Файлы

- `product_types.class.php` - Серверная логика
- `product_types.tpl` - Шаблон страницы

## Зависимости

- View БД: `v_tip_tovara`
- Модуль `product_type` - для отображения страницы типа

## Особенности

1. **Алфавитная группировка:** Типы сгруппированы по первой букве
2. **Счетчики:** Отображение количества товаров каждого типа
3. **Навигация:** Ссылки на страницы типов
