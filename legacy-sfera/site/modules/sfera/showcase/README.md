# Модуль витрины (Showcase)

## Описание

Модуль витрины отображает главную страницу сайта с различными блоками: карусели, популярные категории, товары и т.д.

## Архитектура

### Серверная часть (PHP)

**Файл:** `showcase.class.php`

#### Основные функции

1. **`execute($arr)`** - Главный метод модуля
   - Собирает данные для главной страницы
   - Загружает различные модули через `{mod}` теги
   - Формирует структуру страницы
   - Передает данные в шаблон

#### Структура главной страницы

Главная страница обычно содержит:

1. **Главная карусель** - `main_carousel`
2. **Карусель товаров** - `product_carousel`
3. **Популярные категории** - `popular_categories`
4. **Топ-10 товаров** - `top10_products`
5. **Случайные товары** - `random_products`
6. **Обзоры продукции** - `product_reviews`
7. **Промо-карусель** - `promo_carousel`

### Клиентская часть

**Файл:** `showcase.tpl`

#### Отображение

Главная страница содержит:
- Различные секции с модулями
- Адаптивная верстка для десктопа и мобильных
- Интеграция всех модулей главной страницы

## Использование

### URL структура

```
/
```

Главная страница доступна по корневому URL.

### Модули на главной

Главная страница использует следующие модули:

```smarty
{mod path="sfera/" mod_name="main_carousel"}
{mod path="sfera/" mod_name="product_carousel"}
{mod path="sfera/" mod_name="popular_categories"}
{mod path="sfera/" mod_name="top10_products"}
{mod path="sfera/" mod_name="random_products"}
{mod path="sfera/" mod_name="product_reviews"}
{mod path="sfera/" mod_name="promo_carousel"}
```

## Файлы

- `showcase.class.php` - Серверная логика
- `showcase.tpl` - Шаблон главной страницы
- `d.tpl` - Десктопная версия
- `m.tpl` - Мобильная версия
- `sections/` - Секции главной страницы

## Зависимости

- Все модули главной страницы (см. выше)
- Smarty - для рендеринга шаблонов

## Особенности

1. **Модульность:** Главная страница собирается из отдельных модулей
2. **Адаптивность:** Поддержка десктопной и мобильной версий
3. **Гибкость:** Легко добавлять/удалять модули
4. **Производительность:** Каждый модуль загружается независимо

## Механизм синхронизации с корзиной

Главная страница использует общий механизм синхронизации карточек товаров с корзиной, который реализован в модулях `catalog` и `top10_products`.

### Загружаемые скрипты

На главной странице загружаются следующие JavaScript файлы:

```html
<script src="/assets/sfera/js/catalog.js"></script>
<script src="/assets/sfera/js/top10-slider.js"></script>
```

### Синхронизация карточек TOP-10

Модуль `top10_products` на главной странице автоматически синхронизирует состояние карточек с корзиной:

1. **При загрузке страницы** - карточки отображаются в состоянии, соответствующем текущей корзине (определяется на сервере)
2. **При переключении вкладок** - карточки обновляются через событие `visibilitychange`
3. **При возврате фокуса** - карточки обновляются через событие `window.focus`

Подробнее см. документацию модуля `top10_products`.

### Синхронизация других карточек

Модули `random_products`, `product_carousel` и другие используют общий механизм обновления из `catalog.js`:

- Функция `updateProductCardsFromCart()` обновляет все карточки с классом `.product-card`
- Функция `refreshCartState()` загружает актуальное состояние корзины и обновляет карточки

### Обновление при операциях с корзиной

Все карточки товаров на главной странице автоматически обновляются при:

- Добавлении товара в корзину
- Изменении количества товара в корзине
- Удалении товара из корзины
- Переключении между вкладками браузера
- Возврате фокуса на окно браузера

Это обеспечивает актуальность состояния карточек даже при изменении корзины на другой вкладке.
