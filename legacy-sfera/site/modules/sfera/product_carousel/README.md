# Модуль карусели товаров (Product Carousel)

## Описание

Модуль отображает мини-карусель с товарами на главной странице. Показывает 3 случайных товара с флагом `is_new = 1`.

## Архитектура

### Серверная часть (PHP)

**Файл:** `product_carousel.class.php`

#### Основные функции

1. **`execute($arr)`** - Главный метод модуля
   - Загружает 3 случайных товара из таблицы `products`
   - Фильтрует товары с флагом `is_new = 1`
   - Получает цены из таблицы `prices`
   - Форматирует данные для шаблона
   - Если товаров нет - использует заглушку

#### Получение товаров

```php
$query = "SELECT p.id, p.name, p.picture, pr.price as product_price
          FROM products p
          LEFT JOIN prices pr ON p.id = pr.product_id AND pr.price_type_id = '000000002'
          WHERE p.is_new = 1 
          ORDER BY RAND()
          LIMIT 3";
```

#### Форматирование данных

```php
$products[] = [
    'image' => $image,
    'title' => $row['name'],
    'price' => number_format(round($price), 0, ',', ' ') . ' ₽',
    'link' => '/product/' . $row['id'] . '/'
];
```

#### Заглушка

Если товаров нет:

```php
$products = [
    [
        'image' => '/assets/sfera/img/placeholder.jpg',
        'title' => 'Товары не найдены',
        'price' => '—',
        'link' => '/catalog/'
    ]
];
```

### Клиентская часть

**Файл:** `product_carousel.tpl`

#### Отображение

Карусель обычно содержит:
- 3 карточки товаров
- Изображения товаров
- Названия и цены
- Ссылки на страницы товаров
- Навигационные стрелки

## Использование

### Вызов модуля

```smarty
{mod path="sfera/" mod_name="product_carousel"}
```

## Файлы

- `product_carousel.class.php` - Серверная логика
- `product_carousel.tpl` - Шаблон карусели

## Зависимости

- Таблица `products` - товары
- Таблица `prices` - цены
- JavaScript библиотека для карусели

## Особенности

1. **Случайность:** Товары выбираются случайно через `ORDER BY RAND()`
2. **Новинки:** Показываются только товары с флагом `is_new = 1`
3. **Ограничение:** Всегда 3 товара
4. **Заглушка:** Автоматическая заглушка при отсутствии товаров
