# Модуль страницы тематики (Topic)

## Описание

Модуль отображает страницу с товарами конкретной тематики. Показывает список всех товаров тематики с пагинацией, рейтингами и отзывами.

## Архитектура

### Серверная часть (PHP)

**Файл:** `topic.class.php`

#### Основные функции

1. **`execute($arr)`** - Главный метод модуля
   - Получает ID тематики из GET параметра `topic_id`
   - Получает информацию о тематике из view `v_tematika`
   - Загружает товары этой тематики из таблицы `attributes`
   - Форматирует данные товаров с рейтингами
   - Передает данные в шаблон

2. **`getProductRating($product_id)`** - Получение рейтинга товара
   - Получает SKU товара из `v_products_o_products`
   - Вычисляет средний рейтинг и количество отзывов из `o_reviews`
   - Возвращает массив с `rating` и `reviews_count`

#### Получение информации о тематике

```php
$topicInfo = row("SELECT 
                    id,
                    value AS name,
                    cnt AS count
                  FROM v_tematika
                  WHERE id = $topic_id_safe
                  LIMIT 1");
```

#### Получение товаров тематики

```php
$query = "SELECT DISTINCT p.id, p.name, p.description, p.picture, p.category_id, p.quantity, pr.price as product_price
          FROM products p
          INNER JOIN attributes a ON p.id = a.product_id
          LEFT JOIN prices pr ON p.id = pr.product_id AND pr.price_type_id = '000000002'
          WHERE BINARY a.name = 'Тематика'
            AND BINARY a.value = BINARY '$topic_name_safe'
          ORDER BY p.name ASC";
```

#### Пагинация

- По 20 товаров на страницу
- Групповая пагинация (показ 5 страниц вокруг текущей)

### Клиентская часть

**Файл:** `topic.tpl`

#### Отображение

1. **Заголовок:** "Товары тематики: {название тематики}"
2. **Информация о тематике:**
   - Название тематики
   - Количество товаров
3. **Список товаров:**
   - Карточки товаров с изображениями
   - Название, цена, рейтинг, количество отзывов
   - Кнопки добавления в корзину и избранное
4. **Пагинация:**
   - Навигация по страницам

## Использование

### URL структура

```
/topic/?topic_id={id}
/topic/?topic_id={id}&page=2
```

Пример: `/topic/?topic_id=5&page=1`

### Переход на страницу тематики

Страницы тематик (`/topics/`) содержат ссылки на страницы конкретных тематик. Также ссылки на тематики есть на страницах товаров в характеристиках.

## Файлы

- `topic.class.php` - Серверная логика
- `topic.tpl` - Шаблон страницы тематики

## Зависимости

- View `v_tematika` - список тематик
- Таблица `attributes` - атрибуты товаров (связь товаров с тематиками)
- Таблица `products` - товары
- Таблица `prices` - цены
- View `v_products_o_products` - для получения SKU
- Таблица `o_reviews` - для рейтингов
- Функция `getProductImageUrl()` - для получения изображений

## Особенности

1. **Валидация:** Проверка существования тематики, 404 если не найдена
2. **Рейтинги:** Отображение рейтинга и количества отзывов для каждого товара
3. **Пагинация:** Групповая пагинация для удобной навигации
4. **Интеграция:** Полная интеграция с корзиной и избранным
5. **Поиск:** Поиск товаров по атрибуту "Тематика"
