# Модуль списка тематик (Topics)

## Описание

Модуль отображает список всех тематик в алфавитном порядке с количеством товаров в каждой тематике.

## Архитектура

### Серверная часть (PHP)

**Файл:** `topics.class.php`

#### Основные функции

1. **Получение тематик:**
   - Загружает список тематик из view `v_tematika`
   - Группирует по первой букве названия
   - Сортирует по алфавиту

2. **Группировка:**
   - Группирует тематики по первой букве
   - Создает структуру для отображения

#### Структура данных

```php
$groupedTopics = [
    'А' => [
        ['id' => 1, 'value' => 'Тематика 1', 'cnt' => 10],
        ['id' => 2, 'value' => 'Тематика 2', 'cnt' => 5],
    ],
    'Б' => [
        ['id' => 3, 'value' => 'Тематика 3', 'cnt' => 8],
    ],
    // ...
]
```

### Клиентская часть

**Файл:** `topics.tpl`

#### Отображение

1. **Заголовок:** "Тематики"
2. **Группы тематик:**
   - Заголовок с буквой
   - Сетка тематик
   - Каждая тематика - ссылка на страницу тематики
   - Отображение количества товаров

## Использование

### URL структура

```
/topics/
```

### Переход на страницу тематики

Клик по тематике ведет на `/topic/?topic_id={id}&page=1`

## Файлы

- `topics.class.php` - Серверная логика
- `topics.tpl` - Шаблон страницы

## Зависимости

- View БД: `v_tematika`
- Модуль `topic` - для отображения страницы тематики

## Особенности

1. **Алфавитная группировка:** Тематики сгруппированы по первой букве
2. **Счетчики:** Отображение количества товаров в каждой тематике
3. **Навигация:** Ссылки на страницы тематик
