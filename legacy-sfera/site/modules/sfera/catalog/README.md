# Модуль каталога (Catalog)

## Описание

Модуль каталога отображает список товаров с возможностью фильтрации, сортировки и пагинации.

## Архитектура

### Серверная часть (PHP)

**Файл:** `catalog.class.php`

#### Основные функции

1. **Получение товаров:**
   - Загрузка товаров из таблицы `products`
   - Фильтрация по категории (если указана)
   - Фильтрация по поисковому запросу (если указан)
   - Применение фильтров из модуля `filter`
   - Пагинация результатов

2. **Получение категории:**
   - Загрузка данных категории из таблицы `tree`
   - Формирование breadcrumbs

3. **Фильтрация:**
   - Интеграция с модулем `filter`
   - Фильтры по авторам, сериям, тематикам, типам товаров
   - Применение фильтров через SQL WHERE условия

4. **Пагинация:**
   - Групповая пагинация (показ групп страниц)
   - Расчет количества страниц
   - Формирование ссылок на страницы

#### Структура данных

```php
$products = [
    [
        'id' => '123',
        'name' => 'Название товара',
        'price' => 1000.00,
        'image' => '/path/to/image.jpg',
        'rating' => 4.5,
        'reviews_count' => 10,
        'quantity' => 5,
        // ... другие поля
    ],
    // ... другие товары
]
```

### Клиентская часть (JavaScript)

**Файл:** `assets/sfera/js/catalog.js`

#### Функциональность

1. **Добавление в корзину:**
   - Кнопка "В корзину" на каждой карточке товара
   - После добавления показывается селектор количества
   - Кнопка "Купить всё" устанавливает максимальное количество

2. **Управление количеством:**
   - Кнопки +/- для изменения количества
   - Проверка максимального количества
   - Обновление через AJAX

3. **Добавление в избранное:**
   - Кнопка избранного на каждой карточке
   - Визуальная индикация (заполненное/пустое сердце)

4. **Фильтры:**
   - Применение фильтров через форму
   - Обновление URL при изменении фильтров

## Использование

### URL структура

```
/catalog/                    - Все товары
/catalog/{category_id}/     - Товары категории
/catalog/?page=2            - Страница 2
/catalog/?author_id=123     - Фильтр по автору
```

### Отображение товаров

Каждая карточка товара содержит:

- Изображение товара
- Название (ссылка на страницу товара)
- Рейтинг и количество отзывов
- Цена
- Кнопка "В корзину" / селектор количества
- Кнопка "Купить всё"
- Кнопка избранного

### Фильтры

Модуль интегрирован с модулем `filter`, который предоставляет:

- Фильтр по авторам
- Фильтр по сериям
- Фильтр по тематикам
- Фильтр по типам товаров

## Особенности

1. **Фильтрация:** Гибкая система фильтров через модуль filter
2. **Пагинация:** Групповая пагинация для удобной навигации
3. **Рейтинг:** Отображение рейтинга и количества отзывов на карточках
4. **Корзина:** Полная интеграция с модулем корзины
5. **Избранное:** Интеграция с модулем избранного

## Механизм синхронизации с корзиной

### Определение состояния корзины при загрузке

При рендеринге шаблона (`catalog.tpl`) карточки товаров определяют свое начальное состояние на основе данных корзины, переданных с сервера через переменную `$cart`.

### Клиентское обновление карточек

После загрузки страницы карточки товаров автоматически обновляются при изменении состояния корзины через JavaScript (`assets/sfera/js/catalog.js`).

#### Основные функции обновления

**1. `updateProductCardsFromCart(cartData)`**

Главная функция обновления состояния всех карточек товаров на странице (включая карточки TOP-10):

```javascript
function updateProductCardsFromCart(cartData) {
    // Обрабатывает как обычные карточки, так и карточки TOP-10
    document.querySelectorAll('.product-card, .top10-product-card').forEach(card => {
        const productId = card.querySelector('[data-product-id]')?.getAttribute('data-product-id');
        const isInCart = cartItemIds.includes(productId);
        
        if (isInCart) {
            showProductQuantityControl(productId, quantity);
        } else {
            resetProductCard(productId);
        }
    });
}
```

**2. `showProductQuantityControl(productId, quantity)`**

Показывает контрол количества для товара, который находится в корзине:

```javascript
function showProductQuantityControl(productId, quantity) {
    // Скрывает кнопку "В корзину"
    // Показывает контрол количества
    // Устанавливает текущее количество из корзины
}
```

**3. `resetProductCard(productId)`**

Сбрасывает карточку товара к начальному состоянию (показывает кнопку "В корзину"):

```javascript
function resetProductCard(productId) {
    // Показывает кнопку "В корзину"
    // Скрывает контрол количества
    // Сбрасывает количество к 1
}
```

**4. `refreshCartState()`**

Загружает актуальное состояние корзины с сервера и обновляет все карточки:

```javascript
function refreshCartState() {
    fetch('/cart/?task=get_cart', {
        method: 'POST',
        headers: { 'Content-Type': 'application/x-www-form-urlencoded' }
    })
    .then(response => response.json())
    .then(data => {
        updateProductCardsFromCart(data); // Обновляет все карточки
        cartCounterViewModel.updateCount(data); // Обновляет счетчик
    });
}
```

### Синхронизация между вкладками

Механизм синхронизации обеспечивает актуальность состояния карточек даже при изменении корзины на другой вкладке браузера:

**1. Обновление при переключении вкладок:**

```javascript
document.addEventListener('visibilitychange', function() {
    if (!document.hidden) {
        refreshCartState(); // Загружает актуальное состояние корзины
    }
});
```

**2. Обновление при возврате фокуса:**

В модуле `top10_products` также реализован обработчик `window.addEventListener('focus')` для обновления карточек TOP-10 при возврате фокуса на окно.

### Обновление при операциях с корзиной

Карточки автоматически обновляются при следующих операциях:

1. **Добавление товара в корзину** - после успешного AJAX запроса вызывается `updateProductCardsFromCart(data)`
2. **Изменение количества** - обновление через `updateCartItemQuantity()` с последующим обновлением карточек
3. **Удаление товара** - карточка сбрасывается к состоянию "В корзину" через `resetProductCard()`

### Структура данных корзины

Функции обновления ожидают следующий формат данных корзины:

```javascript
{
    items: {
        "product_id_1": {
            product_amount: 2,    // Количество товара в корзине
            cost: 1000,            // Цена за единицу
            // ... другие поля
        },
        "product_id_2": {
            product_amount: 1,
            cost: 500,
            // ... другие поля
        }
    },
    total_cart_amount: 3,  // Общее количество товаров
    cart_sum: 2500,         // Общая сумма
    // ... другие поля
}
```

### Интеграция с другими модулями

Функции обновления карточек используются в следующих модулях:

- **`top10_products`** - обновление карточек TOP-10 через `updateProductCardsFromCart()`
- **`random_products`** - обновление карточек случайных товаров
- **`author`**, **`seriya`**, **`topic`**, **`product_type`** - обновление карточек в модулях фильтрации

Все эти модули используют общий механизм обновления через `catalog.js`.
