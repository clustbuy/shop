# Модуль страницы товара (Product)

## Описание

Модуль отображает детальную информацию о товаре: изображения, характеристики, описание, отзывы, рейтинг, а также позволяет добавить товар в корзину или избранное.

## Архитектура

### Серверная часть (PHP)

**Файл:** `product.class.php`

#### Основные функции

1. **`execute($arr)`** - Главный метод модуля
   - Получает ID товара из URL через `getDataFromPath()`
   - Загружает данные товара из таблицы `products` с ценой из `prices`
   - Формирует breadcrumbs (хлебные крошки)
   - Загружает атрибуты товара из таблицы `attributes`
   - Загружает изображения из таблицы `o_images`
   - Загружает отзывы и статистику из таблицы `o_reviews`
   - Передает все данные в шаблон

#### Получение данных товара

```php
$product = row("SELECT p.*, pr.price as product_price
               FROM products p
               LEFT JOIN prices pr ON p.id = pr.product_id AND pr.price_type_id = '000000002'
               WHERE BINARY p.id = BINARY '$product_id_safe'");
```

#### Получение атрибутов

Атрибуты загружаются из таблицы `attributes` с исключением служебных полей:

- Новинка
- Отгружать упаковками
- Отображать
- Стандарт
- Тип обложки

Для атрибутов "Серия", "Тематика" и "Тип товара" дополнительно получается ID из соответствующих views для создания ссылок.

#### Получение изображений

1. Проверяется наличие товара в `v_products_o_products`
2. Если найден, получается `product_id` из Ozon
3. Загружаются изображения из `o_images` по `product_id`
4. Изображения формируются как `/o_images/{product_id}/{image_order}.jpg`
5. Главное изображение добавляется первым, если отсутствует в галерее

#### Получение отзывов

1. Получается SKU товара из `v_products_o_products`
2. Загружаются отзывы из `o_reviews` по SKU
3. Вычисляется статистика:
   - Средний рейтинг
   - Общее количество отзывов
   - Распределение по рейтингам (1-5 звезд)

### Клиентская часть (JavaScript)

**Файл:** `assets/sfera/js/product.js`

#### Функциональность

1. **Галерея изображений:**
   - Переключение между изображениями по клику на миниатюры
   - Отображение главного изображения

2. **Добавление в корзину:**
   - Кнопка "В корзину" добавляет товар через `cartViewModel.addItem()`
   - После добавления показывается селектор количества
   - Кнопка "Купить всё" устанавливает максимальное количество

3. **Управление количеством:**
   - Кнопки +/- для изменения количества
   - Проверка максимального количества из БД
   - При количестве = 1 и нажатии "-" товар удаляется из корзины

4. **Добавление в избранное:**
   - Кнопка избранного отправляет запрос на `/favorites/?task=add_item`
   - Обновляет счетчик избранного

#### Интеграция с корзиной

```javascript
// Добавление товара в корзину
document.querySelector('.btn-add-to-cart').addEventListener('click', function() {
    var productId = this.getAttribute('data-product-id');
    var quantity = parseInt(document.getElementById('quantityInput').value) || 1;
    
    if (typeof cartViewModel !== 'undefined' && cartViewModel !== null) {
        cartViewModel.addItem({
            id: productId,
            quantity: quantity
        });
    }
});
```

## Использование

### URL структура

```
/product/{product_id}/
```

Пример: `/product/123456/`

### Отображение данных

1. **Изображения:**
   - Галерея с миниатюрами слева
   - Главное изображение справа
   - Переключение по клику

2. **Характеристики:**
   - Отображаются в две колонки
   - Атрибуты "Серия", "Тематика", "Тип товара" - ссылки на соответствующие страницы

3. **Отзывы:**
   - Отображаются только если есть отзывы
   - Показывается средний рейтинг и количество
   - Распределение по рейтингам
   - Список отзывов с рейтингом и текстом

4. **Действия:**
   - Добавить в корзину
   - Купить всё (максимальное количество)
   - Добавить в избранное
   - Управление количеством (если товар уже в корзине)

## Файлы

- `product.class.php` - Серверная логика
- `product.tpl` - Шаблон страницы товара
- `assets/sfera/js/product.js` - JavaScript функциональность

## Зависимости

- Модуль `cart` - для добавления в корзину
- Модуль `favorites` - для добавления в избранное
- KnockoutJS - для работы с корзиной
- Функция `getProductImageUrl()` - для получения изображения товара

## Особенности

1. **Изображения:** Поддержка галереи из Ozon и локальных изображений
2. **Атрибуты:** Ссылки на страницы серий, тематик и типов товаров
3. **Отзывы:** Интеграция с отзывами из Ozon
4. **Корзина:** Полная интеграция с модулем корзины
5. **Валидация:** Проверка максимального количества товара
