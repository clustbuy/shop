# Модуль страницы автора (Author)

## Описание

Модуль отображает страницу с товарами конкретного автора. Показывает список всех товаров автора с пагинацией, рейтингами и отзывами.

## Архитектура

### Серверная часть (PHP)

**Файл:** `author.class.php`

#### Основные функции

1. **`execute($arr)`** - Главный метод модуля
   - Получает имя автора из GET параметра `author_name`
   - Декодирует URL-encoded строку
   - Проверяет существование автора в таблице `authors`
   - Загружает товары автора с пагинацией
   - Передает данные в шаблон

2. **`getProductRating($product_id)`** - Получение рейтинга товара
   - Получает SKU товара из `v_products_o_products`
   - Вычисляет средний рейтинг и количество отзывов из `o_reviews`
   - Возвращает массив с `rating` и `reviews_count`

#### Получение данных автора

```php
$authorInfo = row("SELECT 
                    author_name,
                    COUNT(*) AS cnt
                  FROM authors
                  WHERE author_name = '$author_name_safe'
                  GROUP BY author_name");
```

#### Получение товаров автора

```php
$query = "SELECT DISTINCT p.id, p.name, p.description, p.picture, p.category_id, p.quantity, pr.price as product_price
          FROM products p
          INNER JOIN authors a ON p.id = a.product_id
          LEFT JOIN prices pr ON p.id = pr.product_id AND pr.price_type_id = '000000002'
          WHERE a.author_name = '$author_name_safe'
          ORDER BY p.name ASC";
```

#### Пагинация

- По 20 товаров на страницу
- Групповая пагинация (показ 5 страниц вокруг текущей)
- Расчет `startPage` и `endPage` для отображения

### Клиентская часть

**Файл:** `author.tpl`

#### Отображение

1. **Заголовок:** "Книги автора: {имя автора}"
2. **Информация об авторе:**
   - Имя автора
   - Количество книг
3. **Список товаров:**
   - Карточки товаров с изображениями
   - Название, цена, рейтинг, количество отзывов
   - Кнопки добавления в корзину и избранное
4. **Пагинация:**
   - Навигация по страницам

## Использование

### URL структура

```
/author/?author_name={имя_автора}
/author/?author_name={имя_автора}&page=2
```

Пример: `/author/?author_name=Пушкин%20Александр%20Сергеевич`

### Переход на страницу автора

Страницы авторов (`/authors/`) содержат ссылки на страницы конкретных авторов.

## Файлы

- `author.class.php` - Серверная логика
- `author.tpl` - Шаблон страницы автора

## Зависимости

- Таблица `authors` - связь товаров с авторами
- Таблица `products` - товары
- Таблица `prices` - цены
- View `v_products_o_products` - для получения SKU
- Таблица `o_reviews` - для рейтингов
- Функция `getProductImageUrl()` - для получения изображений

## Особенности

1. **URL декодирование:** Корректная обработка URL-encoded имен авторов
2. **Валидация:** Проверка существования автора, 404 если не найден
3. **Рейтинги:** Отображение рейтинга и количества отзывов для каждого товара
4. **Пагинация:** Групповая пагинация для удобной навигации
5. **Интеграция:** Полная интеграция с корзиной и избранным
