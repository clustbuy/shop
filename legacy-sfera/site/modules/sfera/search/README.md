# Модуль поиска (Search)

## Описание

Модуль поиска товаров по названию и другим полям.

## Архитектура

### Серверная часть (PHP)

**Файл:** `search.class.php`

#### Основные функции

1. **Поиск товаров:**
   - Получает поисковый запрос из GET параметра `query`
   - Ищет товары в таблице `products` по названию
   - Применяет фильтры (если указаны)
   - Возвращает результаты с пагинацией

2. **Интеграция с каталогом:**
   - Использует ту же логику отображения, что и модуль `catalog`
   - Передает `search_query` в шаблон для отображения

## Использование

### URL структура

```
/search/?query=название товара
/search/?query=название&page=2
```

### Поиск

1. Ввести запрос в поле поиска
2. Нажать "Найти" или Enter
3. Отобразится список найденных товаров

## Файлы

- `search.class.php` - Серверная логика
- `search.tpl` - Шаблон страницы поиска

## Зависимости

- Модуль `catalog` - для отображения результатов
- Таблица `products` - для поиска

## Особенности

1. **Интеграция:** Использует шаблоны и логику каталога
2. **Пагинация:** Поддержка пагинации результатов
3. **Фильтры:** Возможность применения фильтров к результатам поиска
