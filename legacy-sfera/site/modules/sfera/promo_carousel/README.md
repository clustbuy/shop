# Модуль карусели промо-акций (Promo Carousel)

## Описание

Модуль отображает мини-карусель с промо-акциями на главной странице. Показывает промо-баннеры с градиентными фонами и ссылками.

## Архитектура

### Серверная часть (PHP)

**Файл:** `promo_carousel.class.php`

#### Основные функции

1. **`execute($arr)`** - Главный метод модуля
   - В текущей реализации использует статические данные для демонстрации
   - В будущем будет загружать промо-акции из БД
   - Передает данные в шаблон

#### Структура данных (текущая)

```php
$promos = [
    [
        'title' => 'Скидки до 70%',
        'subtitle' => 'На электронику',
        'background' => 'linear-gradient(135deg, #FF0080 0%, #FF1744 100%)',
        'link' => '#'
    ],
    [
        'title' => 'Доставка',
        'subtitle' => 'Бесплатно от 3000₽',
        'background' => 'linear-gradient(135deg, #00A859 0%, #009951 100%)',
        'link' => '#'
    ]
];
```

### Клиентская часть

**Файл:** `promo_carousel.tpl`

#### Отображение

Карусель обычно содержит:
- Промо-баннеры с градиентными фонами
- Заголовки и подзаголовки
- Ссылки на акции
- Навигационные стрелки

## Использование

### Вызов модуля

```smarty
{mod path="sfera/" mod_name="promo_carousel"}
```

### Будущее развитие

Модуль может быть расширен для:
- Загрузки промо-акций из БД
- Управления через админ-панель
- Настройки цветов и изображений
- Активации/деактивации акций

## Файлы

- `promo_carousel.class.php` - Серверная логика
- `promo_carousel.tpl` - Шаблон карусели

## Зависимости

- JavaScript библиотека для карусели

## Особенности

1. **Статические данные:** В текущей версии использует статические данные
2. **Градиенты:** Поддержка CSS градиентов для фонов
3. **Гибкость:** Легко расширяется для работы с БД
