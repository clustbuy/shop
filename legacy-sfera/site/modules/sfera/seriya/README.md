# Модуль страницы серии (Seriya)

## Описание

Модуль отображает страницу с товарами конкретной серии. Показывает список всех товаров серии с пагинацией, рейтингами и отзывами.

## Архитектура

### Серверная часть (PHP)

**Файл:** `seriya.class.php`

#### Основные функции

1. **`execute($arr)`** - Главный метод модуля
   - Получает ID серии из GET параметра `seriya_id`
   - Получает информацию о серии из view `v_seriya`
   - Загружает товары этой серии из таблицы `attributes`
   - Форматирует данные товаров с рейтингами
   - Передает данные в шаблон

2. **`getProductRating($product_id)`** - Получение рейтинга товара
   - Получает SKU товара из `v_products_o_products`
   - Вычисляет средний рейтинг и количество отзывов из `o_reviews`
   - Возвращает массив с `rating` и `reviews_count`

#### Получение информации о серии

```php
$seriyaInfo = row("SELECT 
                    id,
                    value AS name,
                    cnt AS count
                  FROM v_seriya
                  WHERE id = $seriya_id_safe
                  LIMIT 1");
```

#### Получение товаров серии

```php
$query = "SELECT DISTINCT p.id, p.name, p.description, p.picture, p.category_id, p.quantity, pr.price as product_price
          FROM products p
          INNER JOIN attributes a ON p.id = a.product_id
          LEFT JOIN prices pr ON p.id = pr.product_id AND pr.price_type_id = '000000002'
          WHERE BINARY a.name = 'Серия'
            AND BINARY a.value = BINARY '$seriya_name_safe'
          ORDER BY p.name ASC";
```

#### Пагинация

- По 20 товаров на страницу
- Групповая пагинация (показ 5 страниц вокруг текущей)

### Клиентская часть

**Файл:** `seriya.tpl`

#### Отображение

1. **Заголовок:** "Товары серии: {название серии}"
2. **Информация о серии:**
   - Название серии
   - Количество товаров
3. **Список товаров:**
   - Карточки товаров с изображениями
   - Название, цена, рейтинг, количество отзывов
   - Кнопки добавления в корзину и избранное
4. **Пагинация:**
   - Навигация по страницам

## Использование

### URL структура

```
/seriya/?seriya_id={id}
/seriya/?seriya_id={id}&page=2
```

Пример: `/seriya/?seriya_id=5&page=1`

### Переход на страницу серии

Страницы серий (`/series/`) содержат ссылки на страницы конкретных серий. Также ссылки на серии есть на страницах товаров в характеристиках.

## Файлы

- `seriya.class.php` - Серверная логика
- `seriya.tpl` - Шаблон страницы серии

## Зависимости

- View `v_seriya` - список серий
- Таблица `attributes` - атрибуты товаров (связь товаров с сериями)
- Таблица `products` - товары
- Таблица `prices` - цены
- View `v_products_o_products` - для получения SKU
- Таблица `o_reviews` - для рейтингов
- Функция `getProductImageUrl()` - для получения изображений

## Особенности

1. **Валидация:** Проверка существования серии, 404 если не найдена
2. **Рейтинги:** Отображение рейтинга и количества отзывов для каждого товара
3. **Пагинация:** Групповая пагинация для удобной навигации
4. **Интеграция:** Полная интеграция с корзиной и избранным
5. **Поиск:** Поиск товаров по атрибуту "Серия"
