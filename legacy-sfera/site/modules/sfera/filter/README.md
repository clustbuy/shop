# Модуль фильтров (Filter)

## Описание

Модуль фильтров предоставляет боковую панель с фильтрами для каталога товаров: по авторам, сериям, тематикам и типам товаров.

## Архитектура

### Серверная часть (PHP)

**Файл:** `filter.class.php`

#### Основные функции

1. **Получение фильтров:**
   - Загружает список авторов из view `v_author`
   - Загружает список серий из view `v_seriya`
   - Загружает список тематик из view `v_tematika`
   - Загружает список типов товаров из view `v_tip_tovara`

2. **Применение фильтров:**
   - Фильтры передаются в модуль `catalog` через GET параметры
   - Модуль `catalog` применяет фильтры в SQL запросах

#### Структура данных

```php
$filters = [
    'authors' => [
        ['id' => 1, 'name' => 'Автор 1', 'cnt' => 10],
        // ...
    ],
    'series' => [
        ['id' => 1, 'value' => 'Серия 1', 'cnt' => 5],
        // ...
    ],
    'topics' => [
        ['id' => 1, 'value' => 'Тематика 1', 'cnt' => 8],
        // ...
    ],
    'product_types' => [
        ['id' => 1, 'value' => 'Тип 1', 'cnt' => 12],
        // ...
    ]
]
```

### Клиентская часть

**Файл:** `filter.tpl`

#### Компоненты

1. **Форма фильтров:**
   - Чекбоксы для каждого фильтра
   - Группировка по типам (Авторы, Серии, Тематики, Типы товаров)
   - Кнопка "Применить"

2. **Обработка формы:**
   - При отправке формы добавляются GET параметры к URL
   - Перезагрузка страницы с примененными фильтрами

## Использование

### URL структура

```
/catalog/?author_id=123&seriya_id=456&topic_id=789&product_type_id=321
```

### Применение фильтров

1. Пользователь выбирает фильтры в боковой панели
2. Нажимает "Применить"
3. Страница перезагружается с примененными фильтрами
4. Модуль `catalog` фильтрует товары по выбранным критериям

## Файлы

- `filter.class.php` - Серверная логика
- `filter.tpl` - Шаблон панели фильтров

## Зависимости

- Views БД: `v_author`, `v_seriya`, `v_tematika`, `v_tip_tovara`
- Модуль `catalog` - для применения фильтров

## Особенности

1. **Группировка:** Фильтры сгруппированы по типам
2. **Счетчики:** Показ количества товаров для каждого фильтра
3. **Интеграция:** Полная интеграция с модулем каталога
