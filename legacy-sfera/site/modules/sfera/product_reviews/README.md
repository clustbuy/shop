# Модуль обзоров продукции (Product Reviews)

## Описание

Модуль отображает блок обзоров продукции для главной страницы. Обзоры загружаются из таблицы `product_reviews` и отображаются в виде сетки или карусели.

## Архитектура

### Серверная часть (PHP)

**Файл:** `product_reviews.class.php`

#### Основные функции

1. **`execute($arr)`** - Главный метод модуля
   - Загружает первые 10 активных обзоров из таблицы `product_reviews`
   - Фильтрует только активные (`active = 1`)
   - Сортирует по полю `sort`
   - Создает массив с индексами 0-9 для удобного доступа в шаблоне
   - Передает данные в шаблон

#### Получение обзоров

```php
$query = "SELECT id, name, title, image, content, parent_id, sort, active
          FROM product_reviews
          WHERE active = 1
          ORDER BY sort ASC
          LIMIT 10";
```

#### Структура данных

```php
$reviews_array = [
    0 => [
        'id' => 1,
        'name' => 'review1',
        'title' => 'Заголовок обзора',
        'image' => '/assets/img/review1.jpg',
        'content' => 'Текст обзора',
        'sort' => 0
    ],
    1 => [...],
    // ... до 9 элементов
    9 => null  // если обзоров меньше 10
]
```

### Клиентская часть

**Файл:** `product_reviews.tpl`

#### Отображение

Блок обычно содержит:
- Сетку обзоров (до 10 элементов)
- Изображения обзоров
- Заголовки и краткое описание
- Ссылки на полные обзоры

### Управление обзорами

Обзоры управляются через админ-панель:
- Добавление новых обзоров
- Загрузка изображений
- Редактирование контента
- Изменение порядка сортировки
- Активация/деактивация

## Файлы

- `product_reviews.class.php` - Серверная логика
- `product_reviews.tpl` - Шаблон блока

## Зависимости

- Таблица `product_reviews` - хранение обзоров

## Особенности

1. **Ограничение:** Всегда 10 слотов для обзоров (0-9)
2. **Активность:** Отображение только активных обзоров
3. **Сортировка:** Обзоры отображаются в порядке поля `sort`
4. **Иерархия:** Поддержка родительских обзоров через `parent_id`

## Структура таблицы product_reviews

```sql
CREATE TABLE product_reviews (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(255),        -- Системное имя
    title VARCHAR(255),        -- Заголовок обзора
    image VARCHAR(255),        -- URL изображения
    content TEXT,             -- Текст обзора
    parent_id INT,            -- ID родительского обзора
    sort INT DEFAULT 0,       -- Порядок сортировки
    active TINYINT DEFAULT 1  -- Активность
)
```
