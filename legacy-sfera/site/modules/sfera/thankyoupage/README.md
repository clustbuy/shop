# Модуль страницы благодарности (Thank You Page)

## Описание

Модуль отображает страницу подтверждения успешного оформления заказа.

## Архитектура

### Серверная часть (PHP)

**Файл:** `thankyoupage.class.php`

#### Основные функции

1. **`execute($arr)`** - Главный метод модуля
   - Получает `order_code` из GET параметров
   - Загружает данные заказа из таблицы `orders`
   - Передает данные в шаблон

#### Получение данных заказа

```php
$order_code = isset($_GET['order_code']) ? $_GET['order_code'] : null;
$order_info = null;

if ($order_code) {
    $order_info = row("SELECT * FROM orders WHERE order_code = '" . noSQL($order_code) . "' LIMIT 1");
}
```

### Клиентская часть

**Файл:** `thankyoupage.tpl`

#### Отображение

1. **Иконка успеха:** SVG иконка галочки
2. **Заголовок:** "Заказ успешно оформлен!"
3. **Информация о заказе:**
   - Номер заказа (`order_code`)
   - ФИО получателя
   - Телефон
   - Итоговая сумма
4. **Кнопка:** "Вернуться на главную"

## Использование

### URL структура

```
/thankyoupage/?order_code={order_code}
```

Пример: `/thankyoupage/?order_code=ABC123`

### Редирект после заказа

После успешного создания заказа в модуле `checkout` происходит редирект:

```php
echo json_encode([
    'success' => true,
    'redirect' => '/thankyoupage/?order_code=' . $order_keys['order_code']
]);
```

## Файлы

- `thankyoupage.class.php` - Серверная логика
- `thankyoupage.tpl` - Шаблон страницы

## Зависимости

- Модуль `checkout` - для создания заказа
- Таблица `orders` - для получения данных заказа

## Особенности

1. **Простота:** Минималистичный дизайн страницы подтверждения
2. **Информативность:** Отображение ключевой информации о заказе
3. **Навигация:** Кнопка возврата на главную
