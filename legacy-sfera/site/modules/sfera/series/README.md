# Модуль списка серий (Series)

## Описание

Модуль отображает список всех серий в алфавитном порядке с количеством товаров в каждой серии.

## Архитектура

### Серверная часть (PHP)

**Файл:** `series.class.php`

#### Основные функции

1. **Получение серий:**
   - Загружает список серий из view `v_seriya`
   - Группирует по первой букве названия
   - Сортирует по алфавиту

2. **Группировка:**
   - Группирует серии по первой букве
   - Создает структуру для отображения

#### Структура данных

```php
$groupedSeries = [
    'А' => [
        ['id' => 1, 'value' => 'Серия 1', 'cnt' => 10],
        ['id' => 2, 'value' => 'Серия 2', 'cnt' => 5],
    ],
    'Б' => [
        ['id' => 3, 'value' => 'Серия 3', 'cnt' => 8],
    ],
    // ...
]
```

### Клиентская часть

**Файл:** `series.tpl`

#### Отображение

1. **Заголовок:** "Серии"
2. **Группы серий:**
   - Заголовок с буквой
   - Сетка серий
   - Каждая серия - ссылка на страницу серии
   - Отображение количества товаров

## Использование

### URL структура

```
/series/
```

### Переход на страницу серии

Клик по серии ведет на `/seriya/?seriya_id={id}&page=1`

## Файлы

- `series.class.php` - Серверная логика
- `series.tpl` - Шаблон страницы

## Зависимости

- View БД: `v_seriya`
- Модуль `seriya` - для отображения страницы серии

## Особенности

1. **Алфавитная группировка:** Серии сгруппированы по первой букве
2. **Счетчики:** Отображение количества товаров в каждой серии
3. **Навигация:** Ссылки на страницы серий
