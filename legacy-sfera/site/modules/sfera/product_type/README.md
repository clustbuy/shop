# Модуль страницы типа товара (Product Type)

## Описание

Модуль отображает страницу с товарами конкретного типа товара. Показывает список всех товаров указанного типа с пагинацией, рейтингами и отзывами.

## Архитектура

### Серверная часть (PHP)

**Файл:** `product_type.class.php`

#### Основные функции

1. **`execute($arr)`** - Главный метод модуля
   - Получает ID типа товара из GET параметра `product_type_id`
   - Получает информацию о типе из view `v_tip_tovara`
   - Загружает товары этого типа из таблицы `attributes`
   - Форматирует данные товаров с рейтингами
   - Передает данные в шаблон

2. **`getProductRating($product_id)`** - Получение рейтинга товара
   - Получает SKU товара из `v_products_o_products`
   - Вычисляет средний рейтинг и количество отзывов из `o_reviews`
   - Возвращает массив с `rating` и `reviews_count`

#### Получение информации о типе

```php
$productTypeInfo = row("SELECT 
                        id,
                        value AS name,
                        cnt AS count
                      FROM v_tip_tovara
                      WHERE id = $product_type_id_safe
                      LIMIT 1");
```

#### Получение товаров типа

```php
$query = "SELECT DISTINCT p.id, p.name, p.description, p.picture, p.category_id, p.quantity, pr.price as product_price
          FROM products p
          INNER JOIN attributes a ON p.id = a.product_id
          LEFT JOIN prices pr ON p.id = pr.product_id AND pr.price_type_id = '000000002'
          WHERE BINARY a.name = 'Тип товара'
            AND BINARY a.value = BINARY '$product_type_name_safe'
          ORDER BY p.name ASC";
```

#### Пагинация

- По 20 товаров на страницу
- Групповая пагинация (показ 5 страниц вокруг текущей)

### Клиентская часть

**Файл:** `product_type.tpl`

#### Отображение

1. **Заголовок:** "Товары типа: {название типа}"
2. **Информация о типе:**
   - Название типа
   - Количество товаров
3. **Список товаров:**
   - Карточки товаров с изображениями
   - Название, цена, рейтинг, количество отзывов
   - Кнопки добавления в корзину и избранное
4. **Пагинация:**
   - Навигация по страницам

## Использование

### URL структура

```
/product_type/?product_type_id={id}
/product_type/?product_type_id={id}&page=2
```

Пример: `/product_type/?product_type_id=5&page=1`

### Переход на страницу типа

Страницы типов товаров (`/product_types/`) содержат ссылки на страницы конкретных типов.

## Файлы

- `product_type.class.php` - Серверная логика
- `product_type.tpl` - Шаблон страницы типа

## Зависимости

- View `v_tip_tovara` - список типов товаров
- Таблица `attributes` - атрибуты товаров (связь товаров с типами)
- Таблица `products` - товары
- Таблица `prices` - цены
- View `v_products_o_products` - для получения SKU
- Таблица `o_reviews` - для рейтингов
- Функция `getProductImageUrl()` - для получения изображений

## Особенности

1. **Валидация:** Проверка существования типа, 404 если не найден
2. **Рейтинги:** Отображение рейтинга и количества отзывов для каждого товара
3. **Пагинация:** Групповая пагинация для удобной навигации
4. **Интеграция:** Полная интеграция с корзиной и избранным
5. **Поиск:** Поиск товаров по атрибуту "Тип товара"
