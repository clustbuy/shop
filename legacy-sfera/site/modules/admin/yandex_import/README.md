# Модуль импорта из Яндекс (Yandex Import)

## Описание

Модуль импорта пунктов выдачи заказов (ПВЗ) из Яндекс.Доставка из локального JSON файла.

## Архитектура

### Серверная часть (PHP)

**Файл:** `yandex_import.class.php`

#### Основные методы

1. **`import_points()`** - Импорт ПВЗ
   - Читает JSON файл `/upload/response.json`
   - Парсит массив точек выдачи
   - Для каждой точки извлекает:
     - `id` - ID точки
     - `position.latitude` - Широта
     - `position.longitude` - Долгота
     - Весь JSON объект - для сохранения в поле `json`
   - Сохраняет/обновляет в таблицу `points`
   - Логирует все операции

#### Структура таблицы `points`

```sql
CREATE TABLE points (
    id VARCHAR(50),              -- ID точки выдачи
    la DECIMAL(9, 6),           -- Широта
    lo DECIMAL(9, 6),           -- Долгота
    json LONGTEXT               -- Полный JSON объект точки
)
```

#### Структура JSON файла

```json
[
    {
        "id": "000193ce-3765-4fb4-a351-d64a4edd2fa4",
        "name": "Постамат Яндекс Маркет",
        "position": {
            "latitude": 55.670753,
            "longitude": 37.884243
        },
        "address": {
            "full_address": "Люберцы Московская улица 5",
            "comment": "Постамат расположен в магазине продуктов Дикси."
        },
        "contact": {
            "phone": "78002342712"
        }
        // ... другие поля
    }
]
```

### Клиентская часть (ExtJS)

**Файл:** `site/modules/admin/desktop/desktop.tpl` (функция `openYandexImport()`)

#### Компоненты

1. **Окно импорта:**
   - Информационная панель
   - Кнопка "Импорт ПВЗ" в toolbar
   - Блок статуса для отображения результатов

2. **Обработка импорта:**
   - Отправка AJAX запроса на `/admin/yandex_import/?action=import_points`
   - Отображение прогресса
   - Показ статистики после завершения

## Использование

### Подготовка данных

1. Получить JSON файл с ПВЗ от Яндекс.Доставка
2. Сохранить файл как `/upload/response.json`

### Импорт

1. Открыть приложение "Импорт из Яндекс" в админке
2. Нажать кнопку "Импорт ПВЗ"
3. Дождаться завершения импорта
4. Проверить статистику:
   - Обработано записей
   - Добавлено новых
   - Обновлено существующих
   - Ошибок

## Файлы

- `yandex_import.class.php` - Серверная логика
- `/upload/response.json` - JSON файл с ПВЗ
- Лог файлы в `/upload/logs/`

## Зависимости

- PHP JSON extension
- MySQL - таблица `points`
- ExtJS - для UI
- ACL - для проверки прав доступа

## Особенности

1. **Валидация JSON:** Проверка валидности JSON перед обработкой
2. **Обновление существующих:** Если точка уже есть, обновляются координаты и JSON
3. **Логирование:** Детальные логи всех операций
4. **Обработка ошибок:** Пропуск записей с ошибками, продолжение импорта

## Интеграция

Модуль используется на странице checkout для отображения пунктов выдачи на Яндекс.Картах.
