# Модуль управления баннерами (Banners)

## Описание

Модуль управления баннерами позволяет администраторам создавать, редактировать, удалять и сортировать баннеры для отображения на сайте.

## Архитектура

### Серверная часть (PHP)

**Файл:** `banners.class.php`

Модуль работает как REST API, возвращая JSON ответы.

#### Основные методы

1. **`list_banners()`** - Получение списка баннеров
   - Загружает все баннеры из таблицы `banners`
   - Сортирует по полю `sort` (ASC)
   - Возвращает JSON с массивом баннеров

2. **`create_banner()`** - Создание баннера
   - Принимает: `url`, `name`, `title`, `sort`
   - Валидирует обязательные поля (`url`, `name`)
   - Создает запись в таблице `banners`
   - Возвращает JSON с данными созданного баннера

3. **`update_banner()`** - Обновление баннера
   - Принимает: `id` и опциональные поля для обновления
   - Обновляет указанные поля в таблице `banners`
   - Возвращает JSON с результатом

4. **`delete_banner()`** - Удаление баннера
   - Принимает: `id`
   - Удаляет запись из таблицы `banners`
   - Удаляет файл изображения, если он локальный
   - Возвращает JSON с результатом

5. **`upload_banner()`** - Загрузка файла баннера
   - Принимает файл через `$_FILES['file']`
   - Валидирует тип файла (JPEG, PNG, GIF, WebP)
   - Сохраняет в `/assets/sfera/img/banners/`
   - Возвращает JSON с URL загруженного файла

6. **`update_sort()`** - Обновление порядка сортировки
   - Принимает массив ID в порядке сортировки
   - Обновляет поле `sort` для каждого баннера
   - Используется после drag & drop перетаскивания

#### Структура таблицы `banners`

```sql
CREATE TABLE banners (
    id INT PRIMARY KEY AUTO_INCREMENT,
    url VARCHAR(255),        -- URL изображения
    name VARCHAR(255),        -- Название баннера
    title VARCHAR(255),        -- Заголовок
    sort INT DEFAULT 0        -- Порядок сортировки
)
```

#### API Endpoints

Все запросы отправляются на `/admin/banners/?action={action}`:

- `action=list` - GET - Получить список баннеров
- `action=create` - POST - Создать баннер
- `action=update` - POST - Обновить баннер
- `action=delete` - POST - Удалить баннер
- `action=upload` - POST - Загрузить файл
- `action=updateSort` - POST - Обновить порядок

### Клиентская часть (ExtJS)

**Файл:** `site/modules/admin/desktop/desktop.tpl` (функция `openBanners()`)

#### Компоненты

1. **ExtJS Grid:**
   - Отображение списка баннеров
   - Колонки: ID, Порядок, Превью, Название, Заголовок
   - Drag & Drop для изменения порядка
   - Редактирование через двойной клик

2. **Toolbar:**
   - Кнопка "Добавить баннер"
   - Кнопка "Удалить"
   - Кнопка "Обновить"

3. **Окно редактирования:**
   - Поля: URL, Название, Заголовок, Порядок
   - Кнопка загрузки файла
   - Кнопки "Сохранить" и "Отмена"

#### Drag & Drop сортировка

```javascript
viewConfig: {
    plugins: {
        ptype: 'gridviewdragdrop',
        dragGroup: 'bannersDDGroup',
        dropGroup: 'bannersDDGroup'
    },
    listeners: {
        drop: function(node, data, overModel, dropPosition) {
            // Собираем новый порядок ID
            var order = [];
            store.each(function(record) {
                order.push(record.get('id'));
            });
            
            // Отправляем на сервер
            Ext.Ajax.request({
                url: '/admin/banners/?action=updateSort',
                params: {
                    order: JSON.stringify(order)
                },
                success: function(response) {
                    // Обновляем данные из сервера
                    store.load();
                }
            });
        }
    }
}
```

## Использование

### Создание баннера

1. Нажать кнопку "Добавить баннер"
2. Заполнить форму:
   - URL изображения (или загрузить файл)
   - Название
   - Заголовок
   - Порядок сортировки
3. Нажать "Сохранить"

### Изменение порядка

1. Перетащить строку баннера в нужное место
2. Порядок автоматически сохраняется на сервере
3. Данные обновляются из сервера

### Редактирование

1. Двойной клик на строке баннера
2. Изменить нужные поля
3. Нажать "Сохранить"

### Удаление

1. Выбрать баннер(ы)
2. Нажать кнопку "Удалить"
3. Подтвердить удаление

1. **REST API:** Все операции через JSON API
2. **Drag & Drop:** Визуальная сортировка баннеров
3. **Загрузка файлов:** Поддержка загрузки изображений
4. **Автообновление:** Данные обновляются после изменений
5. **Права доступа:** Проверка прав через ACL
