# Модуль авторизации админки (Auth)

## Описание

Модуль авторизации администраторов в админ-панели.

## Архитектура

### Серверная часть (PHP)

**Файл:** `auth.class.php`

#### Основные функции

1. **Проверка авторизации:**
   - Проверяет наличие сессии `$_SESSION['admin_user']`
   - Перенаправляет на страницу входа при отсутствии авторизации

2. **Вход:**
   - Проверяет логин и пароль
   - Создает сессию администратора
   - Перенаправляет на рабочий стол

3. **Выход:**
   - Уничтожает сессию
   - Перенаправляет на страницу входа

## Использование

### Вход в админку

1. Перейти на `/admin/`
2. Ввести логин и пароль
3. Нажать "Войти"

### Выход

1. Нажать кнопку "Выход" в админке
2. Сессия будет уничтожена
3. Перенаправление на страницу входа

## Файлы

- `auth.class.php` - Серверная логика
- `auth.tpl` - Шаблон страницы входа

## Зависимости

- PHP Session
- Таблица пользователей админки

## Особенности

1. **Безопасность:** Проверка авторизации для всех модулей админки
2. **Сессии:** Использование PHP сессий для хранения авторизации
