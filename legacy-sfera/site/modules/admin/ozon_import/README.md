# Модуль импорта из Ozon (Ozon Import)

## Описание

Модуль импорта данных из Ozon API: категории, товары, атрибуты, отзывы и изображения.

## Архитектура

### Серверная часть (PHP)

**Файл:** `ozon_import.class.php`

#### Основные методы

1. **`import_categories()`** - Импорт категорий
   - Получает дерево категорий из Ozon API (`/v1/description-category/tree`)
   - Сохраняет в таблицу `tree`
   - Рекурсивная обработка вложенных категорий

2. **`import_products()`** - Импорт товаров
   - Получает список товаров из Ozon API (`/v3/product/list`)
   - Для каждого товара получает детальную информацию (`/v4/product/info/attributes`)
   - Сохраняет в таблицу `o_products`
   - Создает связи в таблице `products`

3. **`import_attr_names()`** - Импорт названий атрибутов
   - Получает список атрибутов из Ozon API (`/v1/description-category/attribute`)
   - Сохраняет в таблицу `attr_names`

4. **`import_reviews()`** - Импорт отзывов
   - Получает отзывы из Ozon API (`/v1/review/list`)
   - Использует пагинацию через `last_id`
   - Сохраняет в таблицу `o_reviews`
   - Сохраняет прогресс в таблицу `import_reviews_log` для возможности возобновления

5. **`import_images()`** - Импорт изображений
   - Получает записи из таблицы `o_images`
   - Создает папки `/o_images/{product_id}/`
   - Скачивает изображения по URL
   - Сохраняет с именем `{image_order}.jpg`
   - Валидирует изображения перед сохранением

#### API методы

1. **`apiRequest($endpoint, $method, $data)`** - Базовый запрос к Ozon API
   - Обработка ошибок
   - Retry логика при сетевых ошибках
   - Логирование запросов

2. **`apiRequestRaw($endpoint, $method, $data)`** - Запрос с возвратом raw ответа
   - Используется для обработки JSON ошибок

#### Структура данных

**Таблица `o_products`:**
- `product_id` - ID товара из Ozon
- `offer_id` - ID оффера
- `sku` - SKU товара
- `name` - Название
- `primary_image` - URL главного изображения
- И другие поля

**Таблица `o_reviews`:**
- `review_id` - UUID отзыва
- `sku` - SKU товара
- `rating` - Рейтинг (1-5)
- `text` - Текст отзыва
- `date` - Дата отзыва

**Таблица `o_images`:**
- `product_id` - ID товара из Ozon
- `image_url` - URL изображения
- `image_order` - Порядок изображения

### Клиентская часть (ExtJS)

**Файл:** `site/modules/admin/desktop/desktop.tpl` (функция `openOzonImport()`)

#### Компоненты

1. **Панель с кнопками:**
   - "Импорт категорий"
   - "Импорт товаров"
   - "Импорт названий атрибутов"
   - "Импорт отзывов"
   - "Импорт изображений"

2. **Блок статуса:**
   - Отображение прогресса импорта
   - Ссылка на лог файл
   - Статистика (обработано, добавлено, ошибок)

#### Обработка запросов

```javascript
Ext.Ajax.request({
    url: '/admin/ozon_import/?action=import_reviews',
    method: 'GET',
    timeout: 1800000, // 30 минут
    success: function(response) {
        var result = Ext.decode(response.responseText);
        if (result.success) {
            // Отображение статистики
        }
    }
});
```

## Использование

### Импорт категорий

1. Нажать кнопку "Импорт категорий"
2. Дождаться завершения
3. Проверить лог файл при необходимости

### Импорт товаров

1. Нажать кнопку "Импорт товаров"
2. Процесс может занять длительное время
3. Проверить статистику в блоке статуса

### Импорт отзывов

1. Нажать кнопку "Импорт отзывов"
2. Импорт использует пагинацию
3. Прогресс сохраняется для возможности возобновления
4. При ошибке можно повторить - импорт продолжится с последнего `last_id`

### Импорт изображений

1. Нажать кнопку "Импорт изображений"
2. Изображения скачиваются из `o_images`
3. Сохраняются в `/o_images/{product_id}/{image_order}.jpg`
4. Пропускаются уже существующие файлы

## Файлы

- `ozon_import.class.php` - Серверная логика
- `ozon_import.tpl` - Шаблон (не используется, UI в desktop.tpl)
- Лог файлы в `/upload/logs/`

## Зависимости

- Ozon API - для получения данных
- cURL - для HTTP запросов
- MySQL - для сохранения данных
- ExtJS - для UI
- ACL - для проверки прав доступа

## Особенности

1. **Retry логика:** Автоматические повторы при сетевых ошибках
2. **Логирование:** Детальные логи всех операций
3. **Прогресс:** Сохранение прогресса для возобновления импорта
4. **Валидация:** Проверка изображений перед сохранением
5. **Таймауты:** Увеличенные таймауты для длительных операций

## Настройка

### Ozon API

Требуется настройка API ключей в конфигурации:

```php
define('OZON_API_KEY', 'your-api-key');
define('OZON_CLIENT_ID', 'your-client-id');
```

### Таблицы БД

Требуются следующие таблицы:
- `o_products` - товары из Ozon
- `o_reviews` - отзывы
- `o_images` - изображения
- `import_reviews_log` - лог импорта отзывов
- `tree` - категории
- `products` - товары сайта
- `attr_names` - названия атрибутов
