# Модуль рабочего стола админки (Desktop)

## Описание

Модуль рабочего стола админки предоставляет интерфейс управления приложениями администратора в стиле Windows 10.

## Архитектура

### Серверная часть (PHP)

**Файл:** `desktop.class.php`

#### Основные функции

1. **Отображение рабочего стола:**
   - Загружает список доступных приложений
   - Проверяет права доступа через ACL
   - Передает данные в шаблон

### Клиентская часть (ExtJS)

**Файл:** `desktop.tpl`

#### Компоненты

1. **Рабочий стол:**
   - Фон рабочего стола
   - Панель задач внизу
   - Кнопка "Пуск" (Start Menu)

2. **Меню "Пуск":**
   - Список приложений в стиле Windows 10
   - Поиск приложений
   - Группировка приложений
   - Кастомный скроллбар в стиле Windows 10

3. **Окна приложений:**
   - ExtJS окна для каждого приложения
   - Управление через панель задач
   - Минимизация, максимизация, закрытие

4. **Панель задач:**
   - Кнопки открытых окон
   - Индикация активного окна
   - Переключение между окнами

#### Приложения

Модуль управляет следующими приложениями:

- **Управление баннерами** - `openBanners()`
- **Импорт товаров из Ozon** - `openOzonImport()`
- **Импорт из Яндекс** - `openYandexImport()`
- И другие приложения

## Использование

### Открытие приложения

1. Нажать кнопку "Пуск"
2. Выбрать приложение из списка
3. Окно приложения откроется на рабочем столе

### Управление окнами

- **Минимизация:** Нажать кнопку минимизации или кликнуть на кнопку в панели задач
- **Максимизация:** Двойной клик на заголовок окна
- **Закрытие:** Кнопка закрытия или клик на кнопку в панели задач с зажатым Ctrl

### Поиск приложений

1. Открыть меню "Пуск"
2. Начать вводить название приложения
3. Приложения фильтруются автоматически

## Файлы

- `desktop.class.php` - Серверная логика
- `desktop.tpl` - Шаблон рабочего стола

## Зависимости

- ExtJS 4.x - для UI компонентов
- ACL - для проверки прав доступа
- jQuery - для некоторых функций

## Особенности

1. **Windows 10 стиль:** Интерфейс в стиле Windows 10 Settings
2. **Кастомный скроллбар:** Тонкий полупрозрачный скроллбар в меню "Пуск"
3. **Управление окнами:** Полноценное управление окнами через панель задач
4. **Поиск:** Быстрый поиск приложений в меню "Пуск"
